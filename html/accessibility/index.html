Progressive Enhancement

This page website is all about progressive enhancement. The sole purpose of this project is that this website's experience is going to be improved by the use of JavaScript. However the website will run just fine without it, have fun!

What is Progressive Enhancement?

Progressive Enhancement is a designing method where we put the core content first. Just like a cake: we start with the base before we top the cake. This website will run just fine without JavaScript. But enabling JavaScript should improve your experience on this website!
